Day 4: Networking Basics on Linux

1. Network Configuration

Dynamic IP: Assigned by a DHCP server (default in most Ubuntu installations).
Static IP: Manually configured and remains fixed.

Network Interfaces
You can check your interfaces with: ip addr show

Configuring Static IP on Ubuntu
1-ls /etc/netplan/
2-sudo nano /etc/netplan/01-netcfg.yaml
3-Add static IP configuration
4-sudo netplan apply
